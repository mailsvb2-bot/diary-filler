# Заполнитель дневников

Готовая локальная Windows-программа для заполнения Word-дневников `.docx`.

## Как получить готовый `.exe`

1. Откройте вкладку **Actions** в репозитории.
2. Выберите workflow **Build Windows EXE**.
3. Нажмите **Run workflow**.
4. После завершения сборки откройте последний запуск.
5. Внизу скачайте artifact **ZapolnitelDnevnikov-Windows-EXE**.
6. Внутри будет готовый файл:

```text
ZapolnitelDnevnikov.exe
```

Пользователю не нужен Python, pip, PySide6 или python-docx. Он просто запускает `.exe` двойным кликом.

## Что делает программа

- заполняет дневники наблюдения в таблицах Word;
- заполняет 3-й столбец `Месяц/Год`;
- меняет месяц автоматически, когда число месяца во 2-м столбце сбрасывается;
- ставит финальную запись выписки в последний дневник каждого файла;
- сохраняет исходники и создаёт отдельную папку результата;
- пишет отчёт обработки.

## Локальная сборка на Windows

Можно запустить:

```bat
СОБРАТЬ_ONEFILE_EXE.bat
```

Но обычному пользователю это не требуется: для него нужен только готовый artifact из GitHub Actions.
