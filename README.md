# Заполнитель дневников

Локальная Windows-программа для заполнения Word-дневников `.docx`.

## Главное

- лёгкий UI на стандартном Tkinter, без тяжёлых UI-зависимостей;
- поле ФИО пациента;
- дата/месяц поступления: `02.2026` или `10.02.2026`;
- дата выписки: точная дата `ДД.ММ.ГГГГ`;
- заполнение столбца `Месяц/Год` от месяца поступления;
- дата выписки всегда становится последней строкой таблицы;
- финальный дневник на дате выписки заполняется стандартным текстом выписки;
- строки 01.01–09.01 и 01.05–09.05 удаляются;
- подписи из шаблона не пересоздаются и остаются в своих строках;
- исправление мужских/женских окончаний по ФИО пациента;
- удаление дат, номеров и служебных префиксов перед текстами дневников;
- удаление словоформ `испытуемый/испытуемая/испытуемого/испытуемой/...`;
- сохранение результата прямо в выбранную папку;
- в папку сохранения кладётся только итоговый `.docx`.

## Как собрать готовый `.exe` через GitHub Actions

1. Откройте вкладку **Actions**.
2. Выберите workflow **Build Windows EXE**.
3. Нажмите **Run workflow**.
4. После завершения откройте последний запуск.
5. Скачайте artifact **ZapolnitelDnevnikov-Windows-EXE**.
6. Внутри будет готовый файл:

```text
ZapolnitelDnevnikov.exe
```

Конечному пользователю не нужен Python, pip или зависимости. Он запускает `.exe` двойным кликом.

## Локальная сборка на Windows

Для разработчика:

```bat
СОБРАТЬ_ONEFILE_EXE.bat
```

## Локальный запуск из Python

```bat
pip install -r requirements.txt
python src\zapolnitel_dnevnikov_app.py
```
